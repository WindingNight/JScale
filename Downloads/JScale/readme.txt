Aby uruchomić program, wejdź do folderu JScale i wpisz: "node server.js".
Program będzie nasłuchiwał w porcie 3333. Aby zmienić port, zmień wartość zmiennej PORT w pliku server.js.

To start the program, enter JScale directory and type "node server.js". 
The program will be listening at port 3333. To change the port, change the PORT variable in server.js.
